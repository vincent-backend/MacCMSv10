模板插件

安装模板

苹果cms 默认是没有前端模板的，需要你自己制作或者使用第三方模板，有了模板才能算是一个完成的网站，安装模板也非常的简单。

- 模板文件上传到 template 目录下

- 登录后台 系统 => 网站参数配置 => 基本设置

- 网站模板：下拉框选择你刚才上传的模板名称

注意

  • 这里的这里的 模板目录 指的是模板存放 html文件 的目录，默认是 html 这里个为了模板安全一般都会去自定义为别的名称
  • 输入的 模板目录 必须和模板真实 html存放目录 完全一致,否则会导致前台提示模板不存在等错误

插件安装

插件应用可以实现无需 二次开发 苹果cms即可得到php后端功能上的扩展，这样有利于后期cms版本无缝升级。

- 插件应用上传到 addons 目录下，符合标准的插件都会在后台应用市场列出来

- 进入后台应用=>应用=>应用市场=>找到你上传的插件点击启用即可

注意

  • 启用插件后最好在后台右上角点击一下更新缓存
  • 无论是cms版本升级还插件版本升级都是要先将插件禁用，cms版本升级需要禁用所有插件
